HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page using a series of elements.

Basic HTML Structure

Every HTML document follows this basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Page</title>
</head>
<body>
    <h1>Welcome to My Page</h1>
    <p>This is a paragraph.</p>
</body>
</html>

Common HTML Elements

  • <h1> to <h6> - Headings (h1 is largest, h6 is smallest)
  • <p> - Paragraph
  • <a> - Link (anchor)
  • <img> - Image
  • <div> - Division/container
  • <span> - Inline container
  • <ul>, <ol>, <li> - Lists

Example: Creating a Simple Page

<body>
    <header>
        <h1>My Website</h1>
        <nav>
            <a href="index.html">Home</a>
            <a href="about.html">About</a>
        </nav>
    </header>
    
    <main>
        <h2>Welcome</h2>
        <p>This is my first webpage!</p>
        <img src="photo.jpg" alt="A description">
    </main>
    
    <footer>
        <p>© 2025 My Website</p>
    </footer>
</body>

Key Concepts

  • Elements: Defined by tags (opening and closing)
  • Attributes: Provide additional information (e.g., href, src)
  • Semantic HTML: Use meaningful tags that describe content
  • Nesting: Elements can contain other elements

Practice Exercise

Try creating a simple webpage about yourself with:

  • A heading with your name
  • A paragraph describing your interests
  • A list of your hobbies
  • A link to your favorite website
