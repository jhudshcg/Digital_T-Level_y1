Python Basics

What is Python?

Python is a versatile, high-level programming language known for its simplicity and readability. It's widely used in web development, data science, automation, and more.

Your First Python Program

The classic "Hello, World!" program:

print("Hello, World!")

Variables and Data Types

Python supports various data types:

# Numbers
age = 18
price = 9.99

# Strings
name = "Alice"
message = 'Hello, World!'

# Booleans
is_student = True
is_graduated = False

# Lists
hobbies = ["reading", "coding", "gaming"]

# Dictionaries
person = {
    "name": "Alice",
    "age": 18,
    "student": True
}

Basic Operations

# Arithmetic
result = 10 + 5
difference = 10 - 5
product = 10 * 5
quotient = 10 / 5

# String concatenation
greeting = "Hello, " + "World!"

# List operations
fruits = ["apple", "banana"]
fruits.append("orange")  # Add item
print(fruits[0])  # Access item

Control Flow

Make decisions with if statements:

age = 18

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

Loops

Repeat actions with loops:

# For loop
for i in range(5):
    print(i)  # Prints 0 to 4

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

# Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

Functions

Create reusable code blocks:

def greet(name):
    return f"Hello, {name}!"

def add_numbers(a, b):
    return a + b

# Using functions
message = greet("Alice")
print(message)  # Hello, Alice!

sum_result = add_numbers(5, 3)
print(sum_result)  # 8
