Tutorial: Your First Webpage

In this tutorial, you'll learn how to create your very first webpage using HTML. No prior experience needed!

Step 1: Set Up Your Environment

Create a new folder on your computer called "my-first-website". Open a text editor (like Notepad, VS Code, or Sublime Text).

Step 2: Create an HTML File

Create a new file called index.html and save it in your folder.

Step 3: Add the Basic Structure

Type the following code into your file:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    
</body>
</html>

Step 4: Add a Heading

Inside the <body> tags, add a heading:

<h1>Welcome to My First Webpage!</h1>

Step 5: Add a Paragraph

Below the heading, add a paragraph:

<p>This is my first webpage. I'm learning HTML!</p>

Step 6: Add a List

Add a list of things you want to learn:

<h2>Things I Want to Learn:</h2>
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
</ul>

Step 7: Add an Image

Add an image (you can use a URL from the internet):

<img src="https://via.placeholder.com/400x200" alt="Placeholder image">

Step 8: Save and View

Save your file and open it in a web browser. You should see your webpage!

Complete Code

Here's what your complete HTML file should look like:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My First Webpage!</h1>
    <p>This is my first webpage. I'm learning HTML!</p>
    
    <h2>Things I Want to Learn:</h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>
    
    <img src="https://via.placeholder.com/400x200" alt="Placeholder image">
</body>
</html>

Challenge

Now try to enhance your webpage by:

  • Adding more headings and paragraphs
  • Creating a link to your favorite website
  • Adding more images
  • Experimenting with different HTML tags
